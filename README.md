# food_order_analysis
This is a beginner-friendly SQL project designed to strengthen foundational concepts such as filtering, aggregation, and joins.
# Overview
This project focuses on analyzing order and product data using SQL.
The goal is to practice basic to intermediate SQL queries and gain insights related to sales, products, and customer orders.
# Tools used
- MySQL
- MySQL Workbench
# Concepts Covered:
- Filtering using WHERE
- Aggregations (SUM, COUNT, AVG)
- GROUP BY and HAVING
- Simple JOIN
 # Sample Analysis Performed:
 - Total number of order places
- Best selling product
- Most ordered items
- Monthly highest sales
- Total revenue  generated per product
- list of orders placed in last 10 months
